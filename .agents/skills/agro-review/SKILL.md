# Agro-Review Skill

Когда пользователь говорит "проанализируй архитектуру" или "/agro-review", сделай следующее:

1. Прочитай все файлы в `src/` и `lib/`
2. Найди:
   - Циклические зависимости
   - Файлы больше 500 строк
3. Выведи отчет на русском языке в формате:
   ### Проблема 1: [название]
   - Где: файл:строка
   - Почему плохо: [объяснение]
   - Как исправить: [конкретные действия]

4. После отчета спроси: "Хотите, я исправлю эти проблемы?"
