# Готовые DOCX-шаблоны для КФХ

В этой папке лежат готовые шаблоны для загрузки в модуль `Документы` в AgroScope.

## Что внутри

- `vnutrenniy-akt-vypolneniya-polevoy-operacii.docx`
  Внутренний акт по одной выполненной полевой операции.
  Использует шаблонный тип `single_document` и режим генерации `by_operation`.

- `ezhednevnyy-reestr-vypolnennykh-polevykh-rabot.docx`
  Ежедневный реестр выполненных полевых работ.
  Использует шаблонный тип `registry_document` и режим генерации `by_day`.

- `reestr-vypolnennykh-rabot-za-period.docx`
  Реестр выполненных полевых работ за период.
  Использует шаблонный тип `registry_document` и режим генерации `by_period`.

## Почему “внутренний акт”, а не просто “акт выполненных работ”

Для КФХ и сельхозпредприятия нет одного универсального обязательного шаблона акта, одинакового для всех хозяйств. Поэтому в репозитории дан безопасный стартовый вариант:

- как внутренний документ по факту одной операции;
- как шаблон, который можно адаптировать под учетную политику конкретного хозяйства;
- без претензии на единственную официальную форму для всех случаев.

Если хозяйство работает с подрядчиками или внешними заказчиками, формулировки и состав реквизитов обычно нужно отдельно адаптировать под их документооборот.

## Как использовать

1. Откройте раздел `Документы`.
2. Загрузите нужный `.docx`.
3. Выберите тип шаблона:
   - `single_document` для акта по одной операции
   - `registry_document` для реестров
4. Проверьте найденные маркеры и статус валидации.
5. Сформируйте документ по операции, дню или периоду.

## Примечание по маркерам

Все маркеры и их описание приходят с backend-каталога токенов. Если словарь токенов будет расширяться, ориентироваться нужно на UI-раздел `Шпаргалка по маркерам`, а не на этот README.
